# purachase-prediction-app

## Overview
This project is a simple Machine Learning web application that predicts whether a customer will purchase a product based on Age and Salary. It uses Logistic Regression and demonstrates the complete ML workflow from training to deployment.

## Tech Stack
Python, Pandas, Scikit-learn, Flask, Joblib, HTML

## Dataset
The dataset contains:
Age
Estimated Salary
Purchased (Target)

## How It Works
Data preprocessing and train-test split
Feature scaling using StandardScaler
Logistic Regression model training

## Model evaluation
Model saved using Joblib
Flask web app for prediction
